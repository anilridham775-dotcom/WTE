# EcoLoop Pro — Waste-to-Energy ERP

## Deploy to Vercel (Free)

1. Upload this folder to GitHub
2. Connect GitHub repo to Vercel
3. Click Deploy — done!

## Modules Included
- Operations Dashboard
- Generator Registry (24 clients)
- Trip & Lift Plans
- Dustbin Management (Type A / B / C)
- Fill Alerts & Inventory
- Yard Inspection & Certification
- Crushing Unit Monitor
- Boiler & Steam Generation
- Billing & Receipts
- Vehicle Registry (5 vehicles)
- Driver Registry (Aadhaar, DL, Medical, Police)
- Vehicle Cleaning Log
- Preventive & Reactive Maintenance
- Maintenance Schedule
- Tyre Management
- Fuel Management
